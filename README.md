# Awesome AI Skills [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> Curated collection of specialized AI assistant skills for Claude Code and opencode, designed to enhance development workflows with expert domain knowledge and best practices.

AI skills are reusable prompt packages that extend AI assistants' capabilities with specialized knowledge in development frameworks, languages, and workflows. Each skill encapsulates expert guidance, templates, and references to help you work more effectively.

## Development Frameworks

### Full-Stack

- **[convex](convex/)** - Expert Convex development assistant for building full-stack TypeScript apps with real-time sync, server functions, database schema, auth, file storage, and deployment.

### Frontend

- **[anthropic-frontend-design](anthropic-frontend-design/)** - Create distinctive, production-grade frontend interfaces with high design quality. Generates creative, polished code that avoids generic AI aesthetics.
- **[opentui-react](opentui-react/)** - Expert assistance for OpenTUI with React. Covers hooks (useKeyboard, useRenderer, useTimeline), JSX patterns, state management, forms, and testing.
- **[opentui-solid](opentui-solid/)** - Expert assistance for OpenTUI with SolidJS. Covers reactive components, signals, fine-grained reactivity, and JSX patterns.

### Terminal UI

- **[opentui](opentui/)** - Expert assistance for OpenTUI core development using the imperative API with vanilla TypeScript. Covers component creation, layout management, input handling, styling, and animations.
- **[opentui-projects](opentui-projects/)** - Expert assistance for OpenTUI project scaffolding, templates, examples, and learning paths.

### API & Backend

- **[trpc](trpc/)** - Generic tRPC implementation guide. Works with any framework (Next.js, Express, Fastify, Hono, Bun) and any package manager.
- **[openrouter](openrouter/)** - Expert OpenRouter API assistant for unified access to 400+ AI models. Covers chat completions, streaming, tool calling, structured outputs, and embeddings.

### Project Scaffolding

- **[better-t-stack](better-t-stack/)** - Expert guidance for using the Better-T-Stack CLI to scaffold type-safe TypeScript projects with customizable configurations.
