# Awesome AI Skills [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> Curated collection of specialized AI assistant skills for Claude Code, designed to enhance development workflows with expert domain knowledge and best practices.

AI skills are reusable prompt packages that extend Claude Code's capabilities with specialized knowledge in development frameworks, languages, and workflows. Each skill encapsulates expert guidance, templates, and references to help you work more effectively.
