# Contributing

Contributions are welcome! This is a curated list of high-quality AI skills for Claude Code.

## Adding a New Skill

Before suggesting a new skill, please ensure:

- **Quality**: The skill provides clear, actionable guidance
- **Documentation**: Includes comprehensive references and examples
- **Maintenance**: The skill is actively maintained
- **Originality**: Offers unique value not covered by existing skills

## Submission Guidelines

1. **Fork** the repository and create your branch from `main`
2. **Add** the skill to the appropriate category in README.md
3. **Follow** the existing format:
   - GitHub links must end with `#readme`
   - Descriptions start with uppercase and end with period
   - Use objective, non-promotional language
   - Keep descriptions concise and clear

## Format Example

```markdown
- [Skill Name](https://github.com/user/repo#readme) - Brief description of what it does.
```

## Pull Request Process

1. Ensure your description follows the format above
2. Update this section if adding new categories
3. Submit a pull request with a clear title
4. Respond to review feedback promptly

## Review Criteria

Submissions are evaluated based on:

- **Relevance**: Directly applicable to Claude Code users
- **Quality**: Well-documented with examples and references
- **Clarity**: Easy to understand and follow
- **Accuracy**: Technical information is correct
- **Maintenance**: Active development and bug fixes

Thank you for contributing!
