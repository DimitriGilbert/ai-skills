## Three-Skill Strategy

Based on this research, three specialized skills are recommended:

### 1. /opentui (Core Development)
- **Focus:** Imperative API, vanilla TypeScript
- **Users:** Core library users
- **Capabilities:** Component creation, layout, input, styling, animation, debugging

### 2. /opentui-react (React Integration)
- **Focus:** React components, hooks, patterns
- **Users:** React developers
- **Capabilities:** React setup, hooks, state management, forms, testing

### 3. /opentui-projects (Scaffolding & Examples)
- **Focus:** Quick start, learning, templates
- **Users:** Beginners, rapid prototyping
- **Capabilities:** Project templates, examples, learning paths

**See `skill-recommendations.md` for complete specifications.**
