### Important Notes

**Profile Picture Limitations:**
- GitHub profile pictures currently only support:
  - `.png`
  - `.jpeg`
  - `.jpg`
- **NOT supported:** GIF animations for profile pictures
- **Workaround:** Display GIFs in your README instead

**Best Practices:**
- Keep animations lightweight (fast loading)
- Use GIFs for demos and tutorials
- SVG animations for text effects
- Don't overuse animations (can be distracting)
- Test loading times

## Common Issues & Solutions

### Issue 1: Profile Not Showing
**Solution:**
- Make sure repository name exactly matches your username
- Repository must be public
- README.md must be in the root directory

### Issue 2: Stats Not Updating
**Solution:**
- Stats update daily (not real-time)
- Wait 24 hours for changes
- Check your username spelling in URLs

### Issue 3: Images Not Loading
**Solution:**
- Check image URLs are correct
- Use HTTPS for all images
- Verify image permissions (if hosted externally)

### Issue 4: Layout Issues on Mobile
**Solution:**
- Limit widget width to 100%
- Avoid too many side-by-side elements
- Test on multiple devices
